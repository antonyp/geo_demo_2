Courier Bookings

This demonstrator shows how a courier booking system could work and how it could be visualised on a map using Google Maps.

  1. A new booking has just been lodged and is in the queue to be assigned to a courier.
  2. The address is geocoded using the Google Geocoding API so it can be ploted on the map geospatially.
  3. A geospatial operation is performed with this point against courier zones to assign the booking to a courier.
  4. The selected courier is notified of the booking.
  5. The operator can view all bookings and courier zones on the map. They can query each for more details by clicking on the map. Bookings are styled based on status. This provides a live operational view with a geospatial context.
